# 🌍 GoodVibes Africa — Web App

### *Connecting Africa Through Music, Culture & Digital Experiences*

---

# ✨ Overview

**GoodVibes Africa** is a modern African entertainment and lifestyle platform designed to connect creators, artists, listeners, and communities across the continent.

The platform delivers:

* 🎵 Music streaming & discovery
* 🎤 Artist promotion
* 📺 Video & media content
* 💬 Social interaction features
* 🌐 African culture showcase
* 📱 Fully responsive user experience
* ⚡ Fast and modern UI/UX

---

# 🚀 Features

## 🎶 Music Experience

* Stream trending African music
* Artist profiles & portfolios
* Album & playlist support
* Music categories by genre
* Search & recommendations
* Like, share & save tracks

## 📊 Content Management

* Upload music & media
* Manage playlists
* Feature trending artists
* Update homepage content
* Moderate uploaded content

## 🌍 Modern Experience

* Mobile-first design

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/goodvibes-africa.git
```

## 2️⃣ Open Project Folder

```bash
cd goodvibes-africa
```

## 3️⃣ Install Dependencies

### PHP Dependencies

```bash
composer install
```

### JavaScript Dependencies

```bash
npm install
```

---

# ▶️ Run Application

## Start Laravel Server

```bash
php artisan serve
```

## Start Frontend Builder

```bash
npm run dev
```

Application will run at:

```bash
http://127.0.0.1:8000
```
